### What I learned

I learned flex wrap makes the cards responsive without using media queries.

```css
.cards-container {
  font-size: 1rem;
  flex-wrap: wrap;
  justify-content: center;
  gap: 1.25rem;
  margin: 1.25rem;
}
```
